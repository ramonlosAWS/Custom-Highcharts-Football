QuickSight Setup Instructions

  1. Create Parameter: Name: DashboardView, Type: String, Default: SUMMARY
  2. Create Calculated Field: Name: SelectedView, Expression: ${DashboardView}
  3. Field Wells: Add SelectedView to GROUP BY
  4. Add Navigation Actions (one per button):
    • SUMMARY button → Set DashboardView = SUMMARY
    • TEAM NEEDS button → Set DashboardView = TEAMNEEDS
    • 2025 RANKS button → Set DashboardView = 2025RANKS
    • 2026 TRACKER button → Set DashboardView = 2026TRACKER
  5. Conditional Visibility: Use DashboardView parameter in conditional rules to show/hide widgets
