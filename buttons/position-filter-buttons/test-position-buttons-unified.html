Position Filter Buttons - Unified Widget

Current Filter (PositionFilter parameter)
ALL

QuickSight Setup

1. Create parameter: PositionFilter (String, default: ALL)

2. Create calc field on minimal dataset: SelectedPosition = ${PositionFilter}

3. Field Wells: Add SelectedPosition to GROUP BY

4. Add 12 Navigation Actions (one per button), each setting PositionFilter to the button value

5. On target visuals, add calc field: ShowByPosition = ifelse(${PositionFilter}='ALL',1,ifelse(Position=${PositionFilter},1,0))

6. Filter target visuals: ShowByPosition = 1
