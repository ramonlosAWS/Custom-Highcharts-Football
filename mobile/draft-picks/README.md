# Draft Picks — Mobile (1024px)

Mobile version of the Draft Picks widget for 1024px tablet dashboard.

## Layout

| Property | Value |
|----------|-------|
| Chart height | 200px |
| Picks font | 22px Barlow Condensed, weight 700 |
| Label font | 14px |
| Accent bar | 6px wide |

## QuickSight Setup

| Property | Value |
|----------|-------|
| Field Wells | GROUP BY: Current Year Draft Picks (col 0), Next Year Draft Picks (col 1) |
| Filter | Team Abbr (single team) |

## Files

- `quicksight-draft-picks-mobile.json` — QuickSight config
- `test-draft-picks-mobile.html` — Local test file
