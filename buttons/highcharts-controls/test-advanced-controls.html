Advanced Interactive Controls — pushing the limits

Text input via on-screen keyboard, multi-select via parameter encoding, and more creative control patterns.

On-Screen Keyboard HACK

Click letter keys to build a text string character by character. Backspace removes last char. Each click appends to parameter.

⚡ In QuickSight: each key is a separate data point. Navigation action appends clicked letter to a string parameter via calc field logic. Requires a "buffer" approach — see README.
Parameter: ""

Multi-Select Checkboxes NEW

Click items to toggle selection. Parameter stores comma-separated values.

⚡ Uses pipe-delimited parameter string. Calc field on target dataset: contains(${Selections}, fieldValue)
Parameter: "QB|WR|TE"

Chip / Tag Selector NEW

Click chips to toggle. Selected chips get filled, unselected are outlined. Multi-select via encoded parameter.

Parameter: "Speed|Power"

Range Selector (Min / Max) WILD

Two rows of ticks — top sets MIN, bottom sets MAX. Highlighted zone shows the selected range.

Parameter: "3-8"

Priority Ranker WILD

Click ▲/▼ arrows to move items up or down in priority order. Parameter stores ordered list.

Parameter: "Speed,Accuracy,Power,Agility"

Numeric Keypad HACK

Phone-style number pad. Click digits to build a number. Clear and backspace included.

Parameter: "42"

Segmented Gauge NEW

Click segments to set a value. Colored zones indicate ranges (low/med/high).

Parameter: 6

Month/Year Picker WILD

Click month cells to select. Arrow buttons cycle year. Parameter stores YYYY-MM.

Parameter: "2026-03"

Emoji Reaction Bar NEW

Click an emoji to react. Selected emoji scales up with count badge.

Parameter: 👍

Breadcrumb Navigator NEW

Click any breadcrumb level to navigate back. Shows hierarchy path with chevrons.

Parameter: "AFC West"

Sort Direction Control NEW

Click column headers to sort. Click again to toggle ASC/DESC. Active column highlighted.

Parameter: "Rating-DESC"

Volume / Level Bars NEW

Click bars to set level. Bars fill from left. Like an equalizer or volume indicator.
