How This Works

The button bar is rendered outside the QuickSight iframe, in your micro-app's DOM. When clicked:

  1. Your app handles the click event directly (no iframe restrictions)
  2. Use embeddedDashboard.setParameters() to update QuickSight parameters
  3. QuickSight filters update based on the parameter value
  4. Listen for PARAMETERS_CHANGED to sync state if needed
