Interactive Controls for QuickSight Highcharts

All controls use scatter plot + triggerClick → navigation action → set parameter. No JavaScript in QuickSight.

Discrete Slider

Click any tick mark to set value. Track fills to selection.

Parameter: 5

Star Rating

Click a star to rate 1–5. Stars fill up to selection.

Parameter: 3

Increment / Decrement Stepper

Click − or + to adjust value. Each click sets parameter to next/prev step.

Parameter: 5

Toggle Switch

Click to toggle ON/OFF. Visual indicator slides.

Parameter: ON

Color Swatch Picker

Click a color to select theme. Ring highlights selection.

Parameter: Blue

Progress Stepper

Click any step to jump. Completed steps show checkmarks.

Parameter: 2

Heatmap Grid Selector

Click any cell to select a row×column combination.

Parameter: Q2-Product B

Thumbnail Card Selector

Click a card to select. Border and shadow highlight selection.

Parameter: Analytics

Radial / Dial Selector

Click segments on the arc to set value. Filled arc shows progress.

Parameter: 7
